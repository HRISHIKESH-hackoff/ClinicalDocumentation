# 🏥 Clinical Documentation System (Voice Prescription Assistant)

A smart AI-assisted clinical documentation tool that helps doctors generate structured prescriptions using **speech-to-text conversion** and **keyword-based medical extraction**.

---

## 🚀 Overview

With increasing patient load, manual prescription writing becomes time-consuming and error-prone.  
This system enables doctors to **dictate prescriptions using voice**, which are then automatically converted into structured, formatted medical documents.

The system improves:
- Documentation speed ⚡
- Accuracy 🧠
- Patient workflow efficiency 🏥

---

## 🎯 Key Features

- 🎤 Voice-to-Text conversion for medical dictation
- 🧾 Automatic prescription formatting
- 🔍 Keyword extraction from medical conversations
- 🧑‍⚕️ Doctor verification and editing layer
- 📩 Email-based prescription delivery (read-only format)
- 🔐 Secure handling of patient medical data
- 🖥️ Simple GUI built using Tkinter

---

## ⚙️ Workflow

1. Input audio (live or uploaded file)
2. Speech converted to text
3. Medical keywords extracted
4. Structured prescription generated
5. Doctor reviews & signs off
6. Prescription sent to patient via email

---

## 🧠 Tech Stack

- Python 3
- Tkinter (GUI)
- SpeechRecognition / NLP (for voice processing)
- SMTP (for email delivery)
